## Tecnologias
 - [React](https://reactjs.org)
 - [Next.JS](https://nextjs.org/)
 - [styled-components](https://styled-components.com/)


## Instalação das dependências
```bash
yarn install
# ou
npm install
```

## Como executar o projeto

```bash
yarn dev
# ou
npm run dev
```
